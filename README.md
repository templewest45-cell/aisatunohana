# あいさつ桜

児童生徒の挨拶運動で、一定以上の声量を検知するたびに桜の花を咲かせるWebアプリです。

## 使い方

1. ローカルサーバーでこのフォルダを開きます。
2. ブラウザで表示されたら「開始」を押します。
3. マイク使用を許可します。
4. 声量がしきい値を超えて一定時間続くと、挨拶1回としてカウントされ、花が1つ咲きます。
5. 「終了」を押すとCSVとPNG画像がダウンロードされます。

## 主な機能

- マイクによる声量検知
- 声量メーター
- しきい値、連続判定時間、再検知までの時間、満開までの花数の調整
- 挨拶回数と花の数の表示
- CSV保存
- 記録用PNG画像保存
- ブラウザ内の簡易記録一覧

## 注意

マイク入力はブラウザの制限により、通常 `localhost` などの安全な環境で動かす必要があります。
ファイルを直接開くより、ローカルサーバー経由で開いてください。
